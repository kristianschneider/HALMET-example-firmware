# This is a fork of the HALMET Example Firmware

It contains :
- DS18B20 temperature sensors
- NMEA1800 GPS data
- NMEA2000 data relay of Garming GFS10 fuel data to signalk

My main purpose was to get fuelflow data for my boat into SignalK
